Be sure to have a generic understanding of what's actually going on when we start and use an app. The key concepts to take with you from this part of the session are:

  • The idea behind the UIApplication object
  • The use of the AppDelegate
  • The purpose of UIViewController (we will cover much more about it later on)
  • Methods to execute code when our app transitions between execution states
We will cover the super-duper-really-important Delegate pattern and UITableView in the next iOS session! Be sure to practice and come along.
